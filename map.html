Accessible Map Example

In this example, I extend the accessibility of the Google Map API. This example is inspired by the Equal Entry blog post Accessible Maps on the Web.

First Things First

We have to crawl before we can walk and walk before we can run. The initial map I begin with is a basic map with a marker indicating a point on the map. To embed this on the page, I followed the Adding a Google Map with a Marker to Your Website tutorial on the Google Maps API. This example uses some basic markup to embed the map.

HTML:

<div id="map"></div>
<script async defer src="https://maps.googleapis.com/maps/api/js?key=[YOUR KEY]&callback=initMap"></script>
